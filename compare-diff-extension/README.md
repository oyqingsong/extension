# DiffView — 文本差异比较工具

一个 Chrome 扩展，支持 JSON、XML、YAML、纯文本的格式化差异比较，提供并排和内联两种视图。

## 功能概览

### 支持的格式

| 格式 | 说明 |
|------|------|
| **JSON** | 自动格式化后比较，支持忽略 Key 顺序 |
| **XML/HTML** | 自动缩进格式化后比较 |
| **YAML** | 统一换行符和缩进后比较 |
| **纯文本** | 支持空白符规范化 |

### 视图模式

- **并排视图** — 左右两栏对比，原始文本在左、对比文本在右，修改行支持字符级差异高亮
- **内联视图** — 差异行上下排列，新增（绿色）和删除（红色）交替显示

### 核心特性

- **自动类型检测** — 根据文本内容特征自动识别 JSON/XML/YAML/纯文本
- **字符级差异高亮** — 对修改行进一步用 `diffChars` 标记具体变化的字符
- **拖拽文件** — 支持直接拖拽文件到输入区，自动识别类型并填入内容
- **右键菜单** — 选中文本后右键「添加到文本对比」，自动填入左右两侧（两侧都选好后自动打开）
- **输入持久化** — 自动保存输入内容到 `chrome.storage`，重新打开弹窗时恢复

## 安装使用

1. 打开 Chrome，地址栏输入 `chrome://extensions`
2. 开启右上角 **开发者模式**
3. 点击 **加载已解压的扩展程序**，选择 `compare-diff-extension` 目录
4. 在弹窗中输入或拖拽两段文本，点击「比较差异」
5. 或在任意页面选中文本 → 右键 → 「添加到文本对比」

**数据流：**
1. 用户在左右输入区粘贴文本或拖拽文件（或通过右键菜单传入）
2. `Detector.detect()` 自动检测文本类型，也支持手动选择
3. `Formatters.format()` 根据类型对文本进行格式化和标准化
4. `Diff.diffLines()` 执行行级差异比较
5. `Renderer._pairChanges()` 将连续的删除+新增配对为"修改"块
6. 对修改块调用 `Diff.diffChars()` 做字符级差异高亮
7. 渲染并排或内联视图，显示统计信息

## 技术细节

### 字符级差异高亮

对于修改行，不只是标记整行新增/删除，而是进一步用 `diffChars` 比较两行内容：

```
原始：const foo = 'bar'
对比：const foo = 'baz'
→ 字符级高亮：bar（红色） vs baz（绿色）
```

### 文本类型检测

`Detector.detect()` 按以下优先级检测：
1. **JSON** — 以 `{` 或 `[` 开头且 `JSON.parse` 成功
2. **XML/HTML** — 以 `<` 开头且包含闭合标签
3. **YAML** — 包含 `key: value` 模式且不含 HTML 标签
4. **纯文本** — 以上均不匹配

拖拽文件时通过扩展名推断类型：`.json` / `.xml` / `.html` / `.yaml` / `.yml` / `.txt`

### 权限说明

| 权限 | 用途 |
|------|------|
| `activeTab` | 获取当前标签页信息 |
| `contextMenus` | 注册右键菜单「添加到文本对比」 |
| `storage` | 持久化输入内容和右键菜单传递的数据 |
