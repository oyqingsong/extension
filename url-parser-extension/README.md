# URL Parser — URL 解析工具

一个 Chrome 扩展，快速解析 URL 并查看其 Search 参数和 Hash 参数，帮助前端开发者调试接口和页面状态。

## 功能概览

### 解析能力

| 特性 | 说明 |
|------|------|
| **URL 自动补全** | 未带 `https://` 前缀的链接自动补齐 |
| **基本信息** | 显示主机名、端口、路径、完整 URL |
| **Search 参数** | 解析 `?key=value` 查询参数，支持重复 key 检测 |
| **Hash 参数** | 解析 `#key=value` 片段参数 |
| **当前页面** | 一键分析当前标签页 URL |
| **一键复制** | 每个参数值均可单独复制 |

### 功能模块

- **URL 解析** — 粘贴任意 URL 自动解析（支持粘贴后自动触发解析）
- **当前页面** — 切换到该标签时自动加载当前页面的 URL 分析结果

## 安装使用

1. 打开 Chrome，地址栏输入 `chrome://extensions`
2. 开启右上角 **开发者模式**
3. 点击 **加载已解压的扩展程序**，选择 `url-parser-extension` 目录
4. 点击扩展图标，粘贴 URL 或切换到「当前页面」标签

### 核心解析流程

1. 输入 URL → `parseURL()` 用 `new URL()` 标准 API 解析
2. 提取 `url.hostname`、`url.port`、`url.pathname`、`url.href`
3. 遍历 `url.searchParams` 提取所有查询参数
4. 解析 `url.hash` 中 `#key=value` 格式的片段参数
5. 检测重复 Key（同名字段出现多次）并高亮标记
6. 渲染参数表格，每个参数行附带 Key/Value 独立复制按钮

## 技术细节

### URL 自动补全

```js
if (!/^[a-zA-Z]+:\/\//.test(urlString))
  urlString = 'https://' + urlString;
```

### 参数解析特性

- **Search 参数** — 使用 `URL.searchParams` 标准 API，自动处理 `%` 编码解码
- **Hash 参数** — 手动解析 `#key1=val1&key2=val2` 格式（仅在 hash 包含 `=` 时触发）
- **重复 Key 检测** — 统计出现次数，在表头和行级标记重复项
- **粘贴自动解析** — 监听 `paste` 事件，50ms 后自动触发解析
